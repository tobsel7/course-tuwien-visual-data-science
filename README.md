# course-tuwien-visual-data-science
Visualization project exploring real estate price and population dynamics in Finland.

View the [interactive chart](https://tobsel7.github.io/course-tuwien-visual-data-science) to see
population and real estate dynamics in Finland.

Some analytic plots are available in the [Jupyter notebooks](notebook).

The original data is source from the [Statistics Finland](https://pxdata.stat.fi)
page:

 - [Demographic Data](https://pxdata.stat.fi/PxWeb/pxweb/en/StatFin/StatFin__vaerak/statfin_vaerak_pxt_11ra.px)
 - [Real Estate Data](https://pxdata.stat.fi/PxWeb/pxweb/en/StatFin/StatFin__kihi/statfin_kihi_pxt_11jb.px)
